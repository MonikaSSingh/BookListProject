# BookListProject

Book List is a small ASP.Net Core Web Application performing CRUD opeartions along with API using RazorPages and MVC. 

Built With:

Front-End
a.) Bootstrap 
b.) DataTable
c.) Toastr JS
d.) SweetAlert
e.) jQuery

Back-End 
a.) C# 
b.) SQL Server 
c.) Entity framework for Integration with a database
